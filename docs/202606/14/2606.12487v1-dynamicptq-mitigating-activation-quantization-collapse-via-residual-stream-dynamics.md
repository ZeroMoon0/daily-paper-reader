## 摘要
训练后量化（PTQ）对于高效的大语言模型推理至关重要，但当权重、激活和KV缓存均量化至4位精度时，可靠地量化激活仍具挑战性。一个关键难点在于大规模激活，其极值主导了激活范围并放大了量化误差。最先进的方法主要通过基于变换的平滑技术（如正交旋转和仿射缩放）来缓解大规模激活，但忽略了残差流的跨层动态。在本文中，我们展示了大规模激活以相位模式在网络深度上出现和消失，触发大的残差变化。这些变化导致新注入的逐层更新主导4位量化尺度，并削弱历史残差信息。为表征这一行为，我们引入了跳跃比和历史特征信噪比。这表明静态的基于变换的平滑无法完全解决由跨层残差变化引起的动态量化不稳定性。基于此分析，我们提出DynamicPTQ，一种针对相位感知混合精度激活量化的动态训练后量化策略。DynamicPTQ从残差流动态中识别量化敏感层，并仅对这些层分配8位激活精度，同时保持权重、KV缓存和其他激活为4位精度。它可以直接集成到强大的PTQ基线中，如QuaRot、SpinQuant和FlatQuant。在LLaMA-2和LLaMA-3上的实验表明，在W4A4KV4量化下，DynamicPTQ持续改进困惑度和零样本问答性能，同时实现1.05至1.07倍的吞吐量提升，且内存开销适中。这些结果展示了实现鲁棒低比特LLM推理的实用路径。

## Abstract
Post-training quantization (PTQ) is essential for efficient large language model inference, but reliably quantizing activations remains challenging when weights, activations, and KV caches are all quantized to 4-bit precision. A key difficulty lies in massive activations, whose extreme values dominate the activation range and amplify quantization errors. State-of-the-art methods mainly mitigate massive activations through transformation-based smoothing, such as orthogonal rotations and affine scaling, but overlook the cross-layer dynamics of the residual stream. In this paper, we show that massive activations emerge and disappear in a phase-wise pattern across network depth, triggering large residual changes. These changes cause newly injected layer-wise updates to dominate the 4-bit quantization scale and weaken historical residual information. To characterize this behavior, we introduce Jump Ratio and Historical Feature SNR. This suggests that static transformation-based smoothing cannot fully resolve dynamic quantization instability caused by cross-layer residual changes. Based on this analysis, we propose DynamicPTQ, a Dynamic Post-Training Quantization policy for phase-aware mixed-precision activation quantization. DynamicPTQ identifies quantization-sensitive layers from residual-stream dynamics and assigns 8-bit activation precision only to these layers, while keeping weights, KV caches, and other activations in 4-bit precision. It can be directly integrated with strong PTQ baselines such as QuaRot, SpinQuant, and FlatQuant. Experiments on LLaMA-2 and LLaMA-3 show that DynamicPTQ consistently improves perplexity and zero-shot QA performance under W4A4KV4 quantization, while achieving 1.05 to 1.07 times throughput improvement with modest memory overhead. These results demonstrate a practical path toward robust low-bit LLM inference.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：后训练量化（PTQ）对大型语言模型（LLM）的高效推理至关重要，但在将权重、激活和 KV 缓存均压缩至 4 位时，激活量化尤为困难。主要挑战源于“大规模激活”——某些 token 或通道的激活值远大于其他部分，主导了量化范围，严重放大量化误差。
- **背景缺陷**：现有最先进的 PTQ 方法（如 QuaRot、SpinQuant、FlatQuant）通过静态的层内变换（正交旋转、仿射缩放）平滑激活分布，但忽略了残差流的跨层动态。论文揭示：大规模激活随网络深度呈阶段性地出现和消失，导致残差状态中出现大的“跳跃”，使得新注入的层更新主导 4 位量化尺度，进而削弱历史残差信息，造成量化崩溃。
- **整体意义**：论文提出一种从残差流动态角度出发的混合精度激活量化策略，旨在不显著增加成本的前提下提升低比特推理的鲁棒性，为实用化 4 位 LLM 推理提供了新路径。

## 2. 方法论

- **核心思想**：通过残差流动态识别量化敏感层，仅对这些层分配 8 位激活精度，其余层、权重和 KV 缓存保持 4 位，从而在关键区域保护历史残差信息。
- **关键技术细节**：
    - **残差流建模**：定义残差状态 \( \mathbf{x}_t^{(\ell)} \) 和层更新 \( \Delta\mathbf{x}_t^{(\ell)} \)，量化后残差更新为 \( \hat{\mathbf{x}}_t^{(\ell+1)} = Q_b(\hat{\mathbf{x}}_t^{(\ell)} + \Delta\mathbf{x}_t^{(\ell)}) \)。
    - **两个新指标**：
        - **跳跃比**：\( \text{JR}_{\ell} = \frac{1}{T}\sum_t \frac{\|\Delta\mathbf{x}_t^{(\ell)}\|_2}{\|\mathbf{x}_t^{(\ell)}\|_2 + \varepsilon} \)，衡量当前更新相对于历史残差的强度。
        - **历史特征信噪比**：\( \text{SNR}_{\text{hist},\ell} = \frac{1}{T}\sum_t 10\log_{10} \frac{\|\mathbf{x}_t^{(\ell)}\|_2^2}{\|\hat{\mathbf{x}}_t^{(\ell)} - \mathbf{x}_t^{(\ell)}\|_2^2 + \varepsilon} \)，衡量量化后历史残差信息的保留程度。
    - **理论分析**：证明即使施加正交旋转，跳跃比保持不变，且量化误差能量与 \( \|\Delta\mathbf{x}_t^{(\ell)}\|_2^2 / 2^{2b} \) 成正比，因此历史 SNR 受限于跳跃比和位宽。
    - **敏感层选择**：满足 \( \text{JR}_\ell > \tau_\text{JR} \land \text{SNR}_{\text{hist},\ell} < \tau_\text{SNR} \) 的层被选为敏感层。实验发现，对于 L 层密集解码器模型，典型敏感层为前 3 层和最后 1 层，即 S = {1, 2, 3, L}。
    - **算法流程**（Algorithm 1）：
        1. 设置权重和 KV 缓存为 4 位。
        2. 根据敏感层集 S 分配激活精度：敏感层 8 位，其余 4 位。
        3. 应用 PTQ 骨干（如 FlatQuant）对模型进行变换。
        4. 使用分配好的位宽量化权重、激活和 KV 缓存。
        5. 在标定数据上估计量化参数（步长、零点等）。
        6. 构建最终量化模型。
- **与现有基线集成**：可直接集成到 QuaRot、SpinQuant、FlatQuant 中，不改变其权重或 KV 缓存量化方式。

## 6. 主要结论与发现

- **DynamicPTQ 持续改进所有测试的 PTQ 基线**：在 W4A4KV4 设定下，无论是 RTN 还是 GPTQ 权重量化，都能降低困惑度并提升零样本 QA 平均准确率。
- **改进幅度因模型尺度和基线强度而异**：在较小模型或基线较弱的设定下（如 QuaRot+RTN 对 LLaMA-3-8B）改进最显著；在大模型或强基线（如 FlatQuant 对 LLaMA-3-70B）上改进更温和但仍为正。
- **RTN 在应用 DynamicPTQ 后可匹配甚至超越 GPTQ**：说明一旦激活侧瓶颈被缓解，权重侧重建的重要性可能下降。
- **残差流动态解释 MoE 模型的鲁棒性**：MoE 的稀疏路由使层更新幅度更小、历史 SNR 更高，因此更易适应低比特量化。
- **效率分析**：吞吐量提升 1.05×–1.07×，内存开销随批量增大而减小（从 1.18× 降至 1.02×）。

- **轻量级集成**：不修改现有 PTQ 框架，仅需在标定时确定敏感层，易部署。
