## 摘要
稀疏专家混合（MoE）架构通过 top-$k$ 路由仅激活一小部分专家，从而在固定推理预算下扩展大语言模型参数。虽然这保持了因果性并适用于自回归语言模型，但离散的 top-$k$ 运算符不可微，迫使每个输入激活固定数量的专家，导致计算利用低效。我们提出 SoftMoE，用截断的 soft top-$k$ LapSum 松弛替代离散路由，实现专家路由的基于梯度优化。我们进一步参数化每层平均激活专家数量并施加全局预算约束，使模型能跨层学习分配专家容量。SoftMoE 与自回归建模完全兼容，在语言建模和下游任务上达到与稀疏 MoE 相当或更优的性能，同时激活的专家显著减少。值得注意的是，学习到的分配高度非均匀，后层激活更多专家。源代码已公开$^\dagger$。

## Abstract
Sparse Mixture-of-Experts (MoE) architectures enable scaling LLM parameters under a fixed inference budget by activating only a small subset of experts via top-$k$ routing. While this preserves causality and suits autoregressive language models, the discrete top-$k$ operator is not differentiable, forcing a fixed number of active experts per input and resulting in inefficient use of computation. We propose SoftMoE, which replaces discrete routing with a truncated soft top-$k$ LapSum relaxation, allowing gradient-based optimization of expert routing. We further parameterize the mean number of active experts per layer and impose a global budget constraint, enabling the model to learn how to allocate expert capacity across layers. SoftMoE remains fully compatible with autoregressive modeling and achieves performance comparable to or better than sparse MoE on language modeling and downstream tasks, while activating significantly fewer experts. Notably, the learned allocation is highly non-uniform, with later layers activating more experts. The source code is publicly available$^\dagger$.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：稀疏专家混合（MoE）架构通过离散的 top‑k 路由激活少量专家，在固定推理预算下扩展大语言模型参数。但离散 top‑k 操作不可微，导致每输入激活固定数量的专家，计算资源利用低效（如图 1 所示）。同时，无法自适应地在不同层和不同 token 间分配专家容量，训练不稳定。
- **背景**：现有改进路由的方法（如 expert‑choice routing、token mixing、线性分配）往往破坏自回归因果性、增加推理计算代价或导致负载不均衡。
- **研究目标**：设计一种保持自回归兼容性、可微、自适应的路由机制，允许模型学习跨层和跨 token 的专家容量分配，提升计算效率并保持或提升性能。

## 2. 方法论
### 核心思想
- 使用基于 LapSum 的截断 soft top‑k 松弛替代硬 top‑k 路由，实现专家选择的端到端梯度优化。
- 参数化每层平均活跃专家数 \(k_l\)，通过全局预算约束 \(\sum_l k_l = K\) 实现跨层自适应分配。

### 关键技术细节
- **SoftTopK 操作**：给定路由分数 \(r \in \mathbb{R}^n\)，求解 \( \text{LapSum}(x) = \sum_{i=1}^n F_{\text{Lap}}(r_i - x) = k \) 得到阈值 \(\tilde{x}\)，然后用 \(F_{\text{Lap}}(r_i - \tilde{x})\) 作为软选择权重 \(\tilde{p} \in [0,1]^n\)，满足 \(\sum \tilde{p}_i = k\)。LapSum 是闭式解，复杂度 \(O(n)\)。
- **截断操作**：为保持稀疏性，对低于阈值 \(\tau\) 的权重置零（类似 ReLU），梯度仅对活跃专家流动。活跃专家数随输入变化。
- **分配学习**：引入可学习参数 \(\eta \in \mathbb{R}^L\)，通过 \(\pi = \text{softmax}(\eta)\)，\(k_l = \pi_l (K - L) + 1\) 保证 \(k_l \ge 1\) 且总和为 \(K\)，可联合优化。
- **上界保护**：每层最大活跃专家数限制为 \(\lceil k_l \cdot \alpha \rceil\)，\(\alpha \in \{2,4\}\)，防止 OOM。

- **性能与效率**：SoftMoE 在多数配置下达到与稀疏 MoE 相当或更优的语言建模损失和下游准确率，同时平均激活专家数显著减少（例如 SoftMoE*‑1.5 在 ≤2 预算组内激活仅 1.53 vs 稀疏 MoE‑2 的 2.0，损失更低）。

- **迁移效果**：在 HellaSwag、PIQA 上 SoftMoE 普遍优于稀疏 MoE；ARC‑E 上因小测试集方差较大，但仍具竞争力。
- **训练稳定性**：软路由未引入额外不稳定因素（图 4 显示训练曲线平滑）。

## 7. 优点
- **创新性**：首次将 LapSum 软 top‑k 松弛应用于 MoE 路由，实现端到端可微，并允许学习跨层专家预算分配，突破了传统固定分配的限制。
- **自回归兼容**：token 独立处理，无需混合不同位置或不同样本，保持因果性，适合 LLM 推理。
- **效率**：路由开销仅为 \(O(n)\)，远小于专家计算；实际中显著减少活跃专家数，降低 FLOPs。
- **负载均衡**：实验表明负载方差与稀疏 MoE 相近或更低（附录表 5），无需额外复杂策略。
- **清晰的经验发现**：揭示了 MoE 中后层更需要专家容量的规律，为深度感知的架构设计提供指导。
