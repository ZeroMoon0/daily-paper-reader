## 摘要
训练后量化（PTQ）对于高效的大语言模型推理至关重要，但当权值、激活值和KV缓存全部量化至4位精度时，可靠地量化激活值仍然具有挑战性。一个关键困难在于大规模激活值，其极值主导了激活值范围并放大了量化误差。最先进的方法主要通过基于变换的平滑技术（如正交旋转和仿射缩放）来缓解大规模激活值，但忽略了残差流的跨层动力学。在本文中，我们展示了大规模激活值以相位模式在网络深度上出现和消失，引发较大的残差变化。这些变化导致新注入的逐层更新主导4位量化尺度，并削弱历史残差信息。为了刻画这一行为，我们引入了跳跃比和历史特征信噪比。这表明基于静态变换的平滑无法完全解决由跨层残差变化引起的动态量化不稳定性。基于此分析，我们提出了DynamicPTQ，一种针对相位感知混合精度激活量化的动态训练后量化策略。DynamicPTQ从残差流动力学中识别出量化敏感层，并仅对这些层分配8位激活精度，同时将权值、KV缓存和其他激活值保持在4位精度。它可以直接集成到强大的PTQ基线方法中，如QuaRot、SpinQuant和FlatQuant。在LLaMA-2和LLaMA-3上的实验表明，在W4A4KV4量化下，DynamicPTQ持续改善困惑度和零样本QA性能，同时在适度的内存开销下实现了1.05至1.07倍的吞吐量提升。这些结果展示了实现鲁棒低位LLM推理的实用路径。

## Abstract
Post-training quantization (PTQ) is essential for efficient large language model inference, but reliably quantizing activations remains challenging when weights, activations, and KV caches are all quantized to 4-bit precision. A key difficulty lies in massive activations, whose extreme values dominate the activation range and amplify quantization errors. State-of-the-art methods mainly mitigate massive activations through transformation-based smoothing, such as orthogonal rotations and affine scaling, but overlook the cross-layer dynamics of the residual stream. In this paper, we show that massive activations emerge and disappear in a phase-wise pattern across network depth, triggering large residual changes. These changes cause newly injected layer-wise updates to dominate the 4-bit quantization scale and weaken historical residual information. To characterize this behavior, we introduce Jump Ratio and Historical Feature SNR. This suggests that static transformation-based smoothing cannot fully resolve dynamic quantization instability caused by cross-layer residual changes. Based on this analysis, we propose DynamicPTQ, a Dynamic Post-Training Quantization policy for phase-aware mixed-precision activation quantization. DynamicPTQ identifies quantization-sensitive layers from residual-stream dynamics and assigns 8-bit activation precision only to these layers, while keeping weights, KV caches, and other activations in 4-bit precision. It can be directly integrated with strong PTQ baselines such as QuaRot, SpinQuant, and FlatQuant. Experiments on LLaMA-2 and LLaMA-3 show that DynamicPTQ consistently improves perplexity and zero-shot QA performance under W4A4KV4 quantization, while achieving 1.05 to 1.07 times throughput improvement with modest memory overhead. These results demonstrate a practical path toward robust low-bit LLM inference.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：大语言模型在后训练量化（PTQ）中，当权重、激活值和KV缓存全部量化到4-bit时，激活值量化尤其困难。主要原因是存在**大规模激活值（massive activations）**，其极值主导量化范围，放大量化误差，导致量化崩溃。
- **现有方法不足**：主流PTQ方法（如SmoothQuant、QuaRot、SpinQuant、FlatQuant）通过正交旋转、仿射缩放等**静态变换**来平滑激活值分布，但忽略了**残差流的跨层动态**。这些方法将每个Transformer层视为独立、静态的单元，未考虑层间残差更新带来的动态变化。
- **本文发现**：大规模激活值以**分阶段模式**在深度网络上出现和消失（三阶段：早期涌现、中期压缩谷、后期重新分布），引发显著的残差跳跃（residual jump），导致新注入的层更新主导4-bit量化尺度，削弱历史残差信息，从而在阶段边界层造成量化不稳定。
- **核心含义**：静态变换无法消除残差跳变带来的动态量化问题，需要根据残差流动力学动态调整激活精度。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：提出**DynamicPTQ**，一种**相位感知的混合精度激活量化**策略。通过两个新指标识别量化敏感层，仅对这些层使用8-bit激活精度，其余层（权重、KV缓存、其他激活）保持4-bit，从而在保持效率的同时缓解量化崩溃。
- **关键技术细节**：
  - **跳跃比（Jump Ratio, JR）**：衡量当前层更新相对于历史残差状态的强度。公式 \( JR_{\ell,t} = \frac{\|\Delta x_t^{(\ell)}\|_2}{\|x_t^{(\ell)}\|_2 + \varepsilon} \)，层级JR为所有token的平均值。高JR表示强更新。
  - **历史特征信噪比（Historical Feature SNR）**：衡量4-bit量化后历史残差信息的保留程度。公式 \( SNR_{\text{hist},\ell} = \frac{1}{T} \sum_t 10\log_{10} \frac{\|x_t^{(\ell)}\|_2^2}{\|\hat{x}_t^{(\ell)} - x_t^{(\ell)}\|_2^2 + \varepsilon} \)，低SNR表示历史信息扭曲严重。
  - **理论分析**：证明即使应用正交旋转（如Hadamard），由于范数保持，跳跃比不变；4-bit量化误差方差与跳跃比平方成正比，静态变换无法消除残差跳变带来的历史信息损失。
  - **敏感层识别**：通过校准数据计算JR和SNR，选择满足 \( JR_\ell > \tau_{JR} \land SNR_{\text{hist},\ell} < \tau_{SNR} \) 的层。对于密集decoder-only模型，经验性地确定为前几层和最后一层（如LLaMA-2-7B: {1,2,3,32}）。
- **算法流程**（Algorithm 1）：
  1. 设置权重和KV缓存为4-bit精度。
  2. 根据相位感知策略设置激活精度：敏感层8-bit，其余4-bit。
  3. 应用PTQ backbone（如QuaRot、FlatQuant）进行变换，使模型易于量化。
  4. 使用分配好的位宽量化权重、激活和KV缓存。
  5. 在校准数据上估计量化参数（步长、零点、裁剪范围等）。
  6. 构建最终量化模型。

## 6. 主要结论与发现
- DynamicPTQ在W4A4KV4设置下**一致降低困惑度**，尤其在基础基线较弱时提升显著（如LLaMA-3-8B+QuaRot-RTN：WikiText-2从10.60降到9.56）。

- **效率优势**：吞吐量提高1.05×–1.07×，峰值内存仅适度增加（大batch下开销更小）。
- **MoE模型**（如DeepSeek-V2-Lite）由于稀疏路由，残差更新更温和，对低比特量化更鲁棒，从残差流动态角度解释了其更优表现。
- **相位边界层**（前几层和最后一层）是量化敏感层，提升其精度可显著缓解量化崩溃。
- **结论**：低比特激活鲁棒性不仅依赖于静态层内分布，还取决于跨层残差流动态。DynamicPTQ为现有PTQ方法提供了互补的、部署友好的解决方案。

## 7. 优点
- **新颖视角**：首次从残差流跨层动态角度分析低比特激活量化问题，引入跳跃比和历史特征信噪比两个指标，理论证明静态变换的局限。
- **高效实用**：仅少量层使用8-bit激活，其余保持4-bit，吞吐量提升且内存开销可控，适合实际部署。
- **即插即用**：可直接集成到QuaRot、SpinQuant、FlatQuant等主流PTQ基线，无需修改其权重或KV缓存量化。
- **实验全面**：覆盖多种模型家族、规模、数据集、量化基线、推理任务和效率分析，结论稳健。
- **理论支撑**：证明了正交变换下跳跃比不变，以及量化误差与跳跃比平方成正比，为方法提供了理论依据。
