## 摘要
大语言模型（LLM）对长上下文的可扩展性从根本上受限于标准注意力的二次复杂度，这促使采用具有次二次成本的线性注意力机制。为了在长上下文下提升表示能力，近期方法以多状态方式组织记忆。然而，现有的多状态线性注意力方法依赖固定的状态合并策略，无法适应动态变化的令牌重要性，不可逆地掩盖关键令牌，导致长序列上严重的误差累积。针对这一局限，我们提出DLA，一种用于多状态线性注意力的动态记忆建模框架。DLA引入了：(i) 信息感知的动态状态合并，基于令牌级信息变化自适应确定状态边界，在语义转换附近保留高分辨率表示，同时积极总结稳定区域；(ii) 容量受限的记忆建模，通过选择性合并相邻的低信息状态来维护固定大小、按时间顺序的状态缓存，以最小信息损失控制记忆增长。我们在两种不同的线性注意力模型上预训练DLA，并在三个类别的16个数据集上评估。实验结果表明DLA相比最先进方法的优越性。

## Abstract
The scalability of Large Language Models (LLMs) to long contexts is fundamentally constrained by the quadratic complexity of standard attention, motivating the adoption of linear attention mechanisms with sub-quadratic cost. To improve representation capacity under long contexts, recent approaches organize memory in a multi-state manner. However, existing multi-state linear attention methods rely on fixed state merging policies that cannot adapt to dynamically varying token importance, irreversibly obscuring critical tokens and causing severe error accumulation over long sequences. To address this limitation, we propose DLA, a dynamic memory modeling framework for multi-state linear attention. DLA introduces (i) Information-Aware Dynamic State Merging, which adaptively determines state boundaries based on token-level information variation, preserving high-resolution representations around semantic transitions while aggressively summarizing stable regions, and (ii) Capacity-Bounded Memory Modeling, which maintains a fixed-size, chronologically ordered state cache by selectively merging adjacent low-information states to control memory growth with minimal information loss. We pre-train DLA on two different linear attention models and evaluate on 16 datasets across three categories. Experimental results demonstrate the superiority of DLA over state-of-the-art.

## 1. 核心问题与研究动机

- **背景**：大语言模型（LLM）处理长上下文时，标准自注意力的计算和内存复杂度呈二次增长（O(T²)），严重限制了可扩展性。
- **现有方案**：线性注意力通过去除softmax并利用矩阵乘法的可结合性，将复杂度降至O(T)。近期多状态线性注意力方法（如Log-Linear Attention）将历史上下文划分为多个片段并压缩为独立状态，以提升表示能力。
- **核心问题**：现有方法依赖**固定块大小或规则化合并策略**，假设序列中信息密度均匀分布。然而真实序列信息密度非均匀，固定策略无法适应动态变化的令牌重要性，导致关键令牌被过早吸收到粗糙摘要中，且合并操作不可逆，造成误差随序列长度累积。
- **研究目标**：提出一种**信息感知、容量可控**的动态记忆建模框架，使线性注意力在长序列下既能保持高表示质量，又拥有可预测的计算成本。

## 2. 方法论

### 核心思想
- 提出 **动态线性注意力（Dynamic Linear Attention, DLA）**，包含两个关键组件：
  1. **信息感知动态状态合并（Information-Aware Dynamic State Merging）**：根据令牌级信息变化自适应决定状态边界，在语义转换处保持高分辨率，在稳定区域主动压缩。
  2. **容量有界记忆建模（Capacity-Bounded Memory Modeling）**：维护固定大小、按时间排序的状态缓存，当缓存满时选择性合并相邻低信息状态，以最小信息损失控制记忆增长。

### 关键技术细节

#### (1) 信息感知动态状态合并
- 定义**状态信息分数（State Information Score, Iᵗ）**，度量当前令牌状态 sₜ 与最近记忆状态 Sₜ₋₁ 的差异：
  ```
  Iᵗ = ||Sₜ - Sₜ₋₁||_F / (||Sₜ₋₁||_F + ε)
  ```
- 预设阈值τ，若 Iᵗ < τ 则将当前令牌合并到当前记忆状态；若 Iᵗ ≥ τ 则触发新状态创建。
- 训练时使用软门控实现可微分，推理时切换为硬阈值分段。
- **理论依据**：定理3.1证明基于块的近似误差上界受块内异质性控制；推论3.2证明固定块策略在非平稳序列上会引入更大偏差，而自适应策略能对齐语义变化点，最小化异质性项。

#### (2) 容量有界记忆建模
- 状态缓存 M 包含最多 K 个状态，每个状态保存：状态矩阵 S ᵢ、令牌计数 n ᵢ、累计信息分数 Ī ᵢ。
- 新状态产生后尝试追加缓存；若缓存已满（m=K），则选择信息密度最低的相邻状态对（i, i+1）进行合并：
  ```
  (i*, i*+1) = argmin (Īᵢ + Īᵢ₊₁) / (nᵢ + nᵢ₊₁)
  ```
- 合并操作：Sᵢ ← Sᵢ + Sᵢ₊₁，nᵢ ← nᵢ + nᵢ₊₁，Īᵢ ← Īᵢ + Īᵢ₊₁，然后删除被合并的状态。
- 输出计算：对缓存中所有状态进行线性注意力聚合，权重 λₜᵢ 由训练时学习的线性层生成。

1. **DLA在所有任务上一致优于Log-Linear Attention**：常识推理平均提升8%（Mamba-2）和6%（DeltaNet）；检索任务平均提升44%和42%；长上下文任务（RULER）提升显著，如多针任务提升高达350%。

3. **DLA效率更高**：相比Log-Linear Attention，DLA在同等条件下具有更高的吞吐量和更低、更稳定的内存占用，证明信息感知合并的计算开销被更好的压缩效率所抵消。
