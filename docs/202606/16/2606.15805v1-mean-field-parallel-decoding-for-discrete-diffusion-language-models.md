## 摘要
离散扩散语言模型通过并行词元生成实现低延迟解码。然而，基于边际置信度独立选择词元会限制并行有效性：单独看起来可靠的词元在同时更新多个位置时可能形成不兼容的配置。我们提出一种无需训练的解码框架，用于协调这些并行更新。在每次前向传播中，该方法为每个掩码位置分配一个提交分数，并利用模型预测分布中导出的成对交互来优化这些分数。变分松弛法导出一种简单的定点更新，能够在单次前向传播中抑制冲突的同步提交。该机制使解码器能够并行提交更多词元，同时保持有竞争力的生成质量。该方法轻量级，无需辅助模型或重新训练，可直接嵌入现有扩散解码流程而无需修改。在推理和代码生成基准上的实验表明，其在质量-延迟权衡方面具有一致改善。

## Abstract
Discrete diffusion language models enable parallel token generation, offering a pathway to low-latency decoding. However, selecting tokens independently by marginal confidence limits effective parallelism: tokens that appear reliable in isolation can form incompatible configurations when several positions are updated at once. We introduce a training-free decoding framework that coordinates these parallel updates. At each forward pass, the method assigns a commit score to each masked position and refines these scores using pairwise interactions derived from the model's predictive distributions. A variational relaxation yields a simple fixed-point update that suppresses conflicting simultaneous commitments within a single forward pass. This mechanism allows the decoder to commit more tokens in parallel while maintaining competitive generation quality. The method is lightweight, requires no auxiliary model or retraining, and drops into existing diffusion decoding pipelines without modification. Experiments on reasoning and code-generation benchmarks show consistent improvements in the quality-latency trade-off.

# 离散扩散语言模型的平均场并行解码（Mean-Field Parallel Decoding for Discrete Diffusion Language Models）

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：大规模语言模型传统上采用自回归（AR）解码，逐词元生成，导致推理延迟高。非自回归和半自回归模型（如离散扩散语言模型，dLLMs）通过并行生成多个词元来提升吞吐量。然而，dLLMs 的因子化反向策略（独立预测每个掩码位置的分布）忽略了词元间的句法一致性和语义依赖，导致“联合不一致性”（joint inconsistency）：分别看高置信度的词元同时提交时可能相互冲突（例如主语-谓语不一致、冗余重复）。
- **问题定义**：如何在保持生成质量的同时，安全地增加并行提交的词元数量？现有方法（如 DAWN、DEMASK 等）要么依赖辅助模型或训练，要么引入额外计算开销。
- **本文核心思想**：将并行词元提交问题形式化为结构化推断问题，通过变分平均场近似，利用模型自身的预测分布构造成对交互，在单次前向传播内协调提交决策，无需额外训练或辅助模型。

## 2. 方法论

### 核心思想
- 将每个掩码位置是否提交视为二值变量 \(s_i \in \{0,1\}\)，构建成对马尔可夫随机场（MRF），能量函数包含一元势（局部置信度）和成对势（预测分布重叠的惩罚）。
- 通过变分平均场近似得到关于提交强度 \(q_i\) 的定点更新方程，迭代后阈值化得到最终提交集。

### 关键技术细节

1. **一元势（局部置信度）**：  
   \(c_i = \log \pi_i(v^{(1)}_i) - \log \pi_i(v^{(2)}_i)\)，即 top-2 预测概率的对数边际差，值越大表示对最可能词元越确信。

2. **成对势（预测重叠）**：  
   使用 Jensen-Shannon 散度（JSD）衡量两个位置预测分布的相似性，归一化得到 \(D_{ij} = 1 - \text{JSD}(\pi_i, \pi_j) / \ln 2\)，再除以全局最大值。\(D_{ij}\) 高意味着分布相似，应避免同时提交（反铁磁耦合）。

3. **能量函数**：  
   \[
   P(S) \propto \exp\left( \sum_i c_i s_i - \sum_{i<j} D_{ij} s_i s_j \right)
   \]

4. **变分平均场推断**：  
   假设完全因子化近似 \(q(S) = \prod_i q_i^{s_i}(1-q_i)^{1-s_i}\)，最大化 ELBO，导出定点方程：
   \[
   q_i = \sigma\left( c_i - \sum_{j \neq i} D_{ij} q_j \right)
   \]
   其中 \(\sigma\) 是 logistic sigmoid 函数。迭代 \(R\) 次（文中 R=2 足够）得到稳定的 \(q_i\)，然后阈值化（阈值 \(\tau\)）决定提交集 \(C_t = \{i | q_i^{(R)} \geq \tau\}\)。

5. **算法流程**：  
   - 从模型单次前向传播获取对数几率，计算 softmax 得到 \(\pi_i\)；
   - 计算局部置信度 \(c_i\)；
   - 计算成对 JSD 矩阵 \(D\)；
   - 迭代平均场更新（通常 2 次）；
   - 阈值化得到提交集。

### 复杂度
- 成对 JSD 计算复杂度 \(O(m^2 |V|)\)，其中 \(m\) 为当前掩码位置数，但由于块解码限制 \(m\) 且可并行化，开销可忽略。整体仍是单次前向传播。

## 6. 主要结论与发现

- 所提方法在所有设置下均获得最高 TPS，平均加速比达 5.12×（相对 Entropy），且准确率与 Entropy 接近（差距通常在 1-3% 以内）。

- 成对交互（JSD 矩阵）比均匀惩罚更有效；单次迭代（R=1）远不如多次迭代（R=2 足够）。
- 参数 \(\tau\) 和块大小提供平滑的质量-延迟折中，允许用户根据需求调节。

## 7. 优点

- **无需训练/轻量级**：完全基于单次前向传播的模型输出来计算，无需额外模型或重训练，即插即用。
- **理论基础扎实**：将提交选择形式化为能量模型+变分推断，具备可解释性。
- **高效**：平均场迭代仅需 2 步收敛，成对计算可高度并行化，实际开销小。
- **广泛适用**：在多个 dLLM 家族（Dream、LLaDA）和任务（推理、代码）上均有效。
- **可调节性**：提供阈值和块大小两个超参数，易于针对不同延迟-质量需求调整。

## 8. 不足与局限

- **成对计算代价**：虽然可控，但当块大小很大时，\(O(m^2|V|)\) 可能成为瓶颈，需要近似方法（如哈希或采样）。
- **仅考虑成对交互**：未建模高阶依赖（如三词元语法约束），可能遗漏某些冲突模式。
- **代理度量的局限性**：JSD 衡量预测分布重叠，但并非真正的联合条件分布，可能误判某些依赖（如分布相似但语义不同）。
